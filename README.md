
Personal Finance Tracker App
This repository contains the code for building a personal finance tracker application using React and Node.js with MongoDB for database storage.


Project Structure
The project is structured into two main folders:

client: Contains all front-end files using React.
server: Houses the Node.js Express backend.
Authentication
User management and authentication are handled using Clerk, a service that simplifies the implementation of authentication in React applications.

Database
MongoDB Atlas is used for database storage. You will learn how to set up a free MongoDB database using MongoDB Atlas for storing user data and financial information securely.

APIs
The backend provides APIs that enable users to interact with the finance tracker application. This includes CRUD operations for managing financial records and user data.

Getting Started
To run the application locally:

Clone this repository.
Navigate to the client and server folders separately and install dependencies using npm install.
Follow the detailed setup instructions in the respective README files in each folder. <br/>
